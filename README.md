### Why

博客虽然不怎么写，但是写博客的东西却没少折腾。WordPress，HEXO 都瞎搞过，虽然可以选择的主题有很多，但总感觉并不是自己想要的。
后来发现了 vitepress，是基于 vite + vue3 的，内容可以很方便的进行定制。另一点 vue3 的稳定版也出来好久了，但实际开发项目中没机会用到，刚好在这里也可以折腾一下 vue3。
